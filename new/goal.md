Project Goal
------------
Build a reproducible Debian-based operating system profile
designed for predictable networking, explicit system behavior,
and modular security workflows.

Core Principles
---------------
- Headless-first, GUI on demand
- Explicit networking, no automatic conflicts
- Minimal default services
- Clear privilege boundaries
- Rebuildable by scripts

Core Capabilities
-----------------
1. Role-based network interfaces
2. Predictable routing and forwarding
3. DNS/DHCP available as optional services
4. Minimal, explainable service set
5. Explicit privilege escalation
6. Self-inspection and auditing tools

Optional Profiles
-----------------
- CTF tools profile
- Infrastructure/lab profile
- GUI tools profile
- Experimental profile
